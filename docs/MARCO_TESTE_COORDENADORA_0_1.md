# Marco: Teste Coordenadora 0.1

Data: 31/05/2026

Este marco congela a versão **Teste Coordenadora 0.1** do CoordFlow.

## Propósito desta versão

Esta versão existe para **observação assistida com coordenadora pedagógica real**.

Ela foi estabilizada para validar uso, fluxo e compreensão operacional em contexto real de trabalho escolar.

## Diretriz de uso

Esta versão **não** foi criada para expansão de produto nesta etapa.

Não é momento de:
- adicionar novos módulos;
- integrar backend real;
- sofisticar IA;
- redesenhar o produto.

## Escopo funcional mantido para o teste

- Mesa Central;
- criação e conclusão de demandas;
- Pautas & Reuniões;
- encaminhamentos;
- Histórico Institucional.

## Regra de decisão desta fase

As próximas decisões devem partir da observação assistida e do aprendizado de uso real, preservando a simplicidade do protótipo como instrumento de teste.
